# Generador de Códigos QR Gratis Online

Aplicación web gratuita para **crear códigos QR** a partir de enlaces URL o texto, y descargarlos como imagen PNG al instante. Ideal para menús de restaurante digitales, catálogos de tienda, tarjetas de presentación o compartir enlaces de WhatsApp.

## ✨ Características

- **Conversión instantánea** de URL o texto a imagen QR.
- **Descarga en PNG** en alta resolución (150px a 500px).
- **Historial local** de los últimos 5 códigos generados (persistencia en LocalStorage).
- **100% gratis y privado**: no se envían datos a servidores propios.
- **Diseño Mobile-First** optimizado para Core Web Vitals.
- Sin registro ni instalación: funciona directamente en el navegador.

## 🚀 Uso

1. Pega la URL o texto en el campo correspondiente.
2. Selecciona el tamaño del QR (opcional).
3. Haz clic en **"Generar Código QR"**.
4. Descarga la imagen con el botón **"Descargar PNG"**.

> Los datos se guardan únicamente en tu navegador (LocalStorage), lo que garantiza privacidad total.

## 🛠️ Tecnologías

- HTML5 semántico
- Tailwind CSS (vía CDN)
- Vanilla JavaScript
- API pública `api.qrserver.com` para renderizado QR
- LocalStorage para persistencia
